Detailed Description

Class to interface an MF52 thermistor device in a voltage divider circuit.

Handles "communication" with an MF52 thermistor by using a basic voltage divider. You only need the thermistor's beta value and the resistance at room temperature (R25). Use the part number or the datasheet (http://radio-hobby.org/uploads/datasheets/mf/mf52[1].pdf) to acquire the necessary information. It is assumed that supply and reference voltage are the same. Value of the second resistor determines accuracy of the readings.
