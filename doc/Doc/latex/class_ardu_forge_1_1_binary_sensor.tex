\doxysubsection{Detailed Description}
This class reads a binary sensor and debounces its input. 

Binary sensors such as switches or reed sensors can have a L\+OW or H\+I\+GH state but can deliver a noisy signal when they switch states. Therefore the signal needs to be debounced. This class debounces the signal and returns L\+OW or H\+I\+GH only if the signal is stable (-\/1 in case of an unstable signal). A signal is considered stable if it does not change a certain number times with a delay between each read.
