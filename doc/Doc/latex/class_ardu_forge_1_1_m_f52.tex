\doxysubsection{Detailed Description}
Class to interface an \mbox{\hyperlink{class_ardu_forge_1_1_m_f52}{M\+F52}} thermistor device in a voltage divider circuit. 

Handles \char`\"{}communication\char`\"{} with an \mbox{\hyperlink{class_ardu_forge_1_1_m_f52}{M\+F52}} thermistor by using a basic voltage divider. You only need the thermistor\textquotesingle{}s beta value and the resistance at room temperature (R25). Use the part number or the datasheet (\href{http://radio-hobby.org/uploads/datasheets/mf/mf52}{\texttt{ http\+://radio-\/hobby.\+org/uploads/datasheets/mf/mf52}}\mbox{[}1\mbox{]}.pdf) to acquire the necessary information. It is assumed that supply and reference voltage are the same.
